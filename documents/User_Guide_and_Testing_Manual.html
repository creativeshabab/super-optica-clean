Operational User Guide & Testing Manual

Overview

This document provides a comprehensive overview of the Optical ERP system's workflows, specifically designed for testing Multi-Tenancy, Inventory Distribution, and validating Real-World Use Cases.

Part 1: System Hierarchy & Concepts

Understanding the layers is crucial for testing:

Super Admin

Platform Owner. Can create Tenants (Optical Chains).

Tenant Admin

Business Owner. Manages a specific chain, users, and products.

Store User

Location Staff. Cashiers, Inventory Managers, Sales.

Scenario 1: Onboarding

Onboarding a New Tenant

Use Case: A new client, "Vista Vision", subscribes to your ERP platform.

Step-by-Step Flow
  1. Login as Super Admin
    Go to SaaS Management panel.
  2. Create Tenant
    Click "Add New Tenant". Enter Company Name ("Vista Vision"), Domain, and Admin Email.
  3. Verify Isolation
    Logout. Login as the new Tenant Admin. Verify the dashboard is empty (0 sales) even if other tenants have data.
Scenario 2: Inventory Setup

Product & Inventory Setup

Use Case: "Vista Vision" buys 100 units of "Ray-Ban Aviators" for their Central Warehouse.

1. Go to Inventory > Warehouses → Create "Central Warehouse".
2. Go to Settings > Stores → Create "Downtown Branch" and "Mall Branch".

Go to Inventory > Products. Add "Ray-Ban Aviator" (SKU: RB-AV-001).
Note: Stock starts at 0.

Go to Inventory > Purchase Orders > New GRN.
Select "Central Warehouse" as destination. Add 100 Qty.
Result: Central Warehouse = 100 units. Stores = 0 units.
Scenario 3: Distribution

Inventory Transfer Flow

Use Case: Move 20 units from Central Warehouse to Downtown Branch.

Sender (Warehouse Owner)
  • Go to Inventory > New Transfer
  • From: Central Warehouse
  • To: Downtown Branch
  • Qty: 20
  • Action: Send
Receiver (Store Manager)
  • Login as Store Manager
  • Go to Incoming Transfers
  • See pending 20 units
  • Action: Accept/Receive
Result: Warehouse Stock drops to 80. Store Stock rises to 20. Total is still 100.
Scenario 4: Sales (POS)

The Sales Cycle

Use Case: Customer buys 1 Ray-Ban at Downtown Branch.

1. Navgiate to POS.
2. Scan RB-AV-001.
3. Checkout and Collect Payment.
4. Verification: Go to Stock View. Downtown Branch should now have 19 units.
