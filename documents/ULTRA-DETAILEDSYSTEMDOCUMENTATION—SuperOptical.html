1. Project Overview & Architecture

Super Optical is a professional optical retail ERP and eCommerce storefront. It combines physical clinic service management (eye tests) with digital commerce (prescription glasses/lenses).

Core Engine

Built with a modular PHP framework using a live dynamic theme injection system.

Data Architecture

Relational MySQL with JSON fields for technical specifications and multi-tier categories.

Auth Logic

Secure passwordless system (Email OTP) for high-conversion user onboarding.

Service Core (Appointments)

Multi-step wizard located at book_appointment.php.

Step 1: Date

Selection of available days. Prevents back-dated entries.

Step 2: Slots

Dynamic 30-min window generation. Checks appointments table for existing bookings to hide full slots.

Step 3: Identity

Collects Name, Phone, and Email. Performs pattern-based validation.

3. End-to-End User Journey

PHASE 1

Awareness & Auth

9. Edge Cases & Safety

Proactive handling of critical failures.

  • Payment Interruption: Webhook/Signature verification logs partial payments to ensure no double-charging.
  • Stock Conflicts: Database-level collision check prevents two users from checking out the same final SKU simultaneously.
  • Session Expiry: Graceful redirection with a "Please Login Again" toast.
