3. Core Logic Deep-Dive

Atomic Stock Governance

Prevents "Overselling" during high-traffic sales events.

STEP 1: Start DB Transaction
STEP 2: Check current stock vs order quantity
STEP 3: Execute decrement query
STEP 4: If (stock < threshold): Dispatch Admin Alert
STEP 5: Commit Transaction

Dynamic Theming (SASS Substitution)

Translates Admin Dashboard settings into runtime CSS.

Fetch settings -> Parse --primary-color -> Inject into <style> head block.

4. Security Protocols

Encryption

OpenSSL AES-256-CBC is used for all third-party API keys stored in DB. Decryption is restricted to the IntegrationManager singleton.

XSS/Injection

Deep sanitization whitelists safe HTML for product specifications while stripping all on* events and <script> tags.
