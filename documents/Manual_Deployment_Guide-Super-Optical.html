1Update Configuration (The Bridge)

This is the most critical step to ensure your site connects to the live database.

  1. Go to your Live Server file manager (inside config/).
  2. Rename the existing db.php to db.php.old.
  3. Upload your NEW local config/db.php file to the server.
  4. Create a new file on the server named config/database.production.php.
  5. Paste your live database credentials into it:
<?php
$host = 'localhost';
$username = 'YOUR_LIVE_DB_USER';
$password = 'YOUR_LIVE_DB_PASSWORD';
$db_name = 'YOUR_LIVE_DB_NAME';
?>

2Upload & Overwrite Files

Upload the new files from your local computer to the server. Overwrite existing files.

3Apply Database Changes

  1. Open PHPMyAdmin for your live database.
  2. Go to the SQL tab.
  3. Open your local file sql/schema_improvements.sql using a text editor.
  4. Copy the SQL code and paste it into PHPMyAdmin.
  5. Click Go.

Ignore "duplicate" errors if the columns already exist.
