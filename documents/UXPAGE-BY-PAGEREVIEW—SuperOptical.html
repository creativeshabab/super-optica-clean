VIEWPORT: HOME

Home Page Audit

The core landing experience focused on "Trust" and "Premium Selection."

Visual Strengths
  • Dynamic Hero Slider: High-impact visuals (Lenskart style) drive immediate engagement.
  • Luxe Trust Badges: Prominent display of "Originality" and "Fast Shipping" builds instant authority.
  • Categorical Pills: Clean mobile navigation for the 5 core categories reduces cognitive load.
UX Weaknesses
  • Scroll Depth: The "Featured Products" section is too far down; should be closer to the fold.
  • Promotion Overload: Having 3 promo banners in a row can lead to "Banner Blindness."
UX FIX: Above-the-fold Optimization

Introduce a small horizontal "Best Sellers" marquee directly under the Hero slider to convert high-intent users immediately.

VIEWPORT: CATALOG

Shop Grid Audit

The high-density product discovery engine.

Visual Strengths
  • Card Density: 4-cols on desktop provides a professional "Mega-store" feel.
  • Price Transparency: Striking MRP vs. Offer Price is visually clear and boosts FOMO.
  • Hover states: Elegant card lift and drop-shadow change provide excellent feedback.
UX Weaknesses
  • Zero Filter States: If a user selects "Cat Eye" and nothing is found, there is no "Clear All" button.
  • Sorting UX: The sort dropdown is buried; should be prominent next to category pills.
UX FIX: Predictive Product Discovery

Add "Shape Filters" (Square, Round, Aviator) with small visual icons instead of just text-based category names.

VIEWPORT: DETAIL

Product Page Audit

The conversion engine where sales happen.

Visual Strengths
  • Sticky Actions: "Add to Cart" and "Buy Now" stay prominent on mobile.
  • Variant Swatches: Color-coded variant circles make selection intuitive.
  • Breadcrumb logic: Clear hierarchical path back to parent categories.
UX Weaknesses
  • Gallery Navigation: Thumbnails can be too small on medium mobile screens.
  • Information Density: "Key Features" list is often too long; needs an accordion for clean UI.
UX FIX: Micro-Interactions

Add a "Haptic-style" bounce animation when the variant color is changed to confirm the user's selection.

VIEWPORT: TRANSACTION

Cart & Checkout Audit

The final steps of the customer journey.

Visual Strengths
  • Stepper UI: The "Shipping -> Payment -> Success" stepper reduces checkout anxiety.
  • Coupon Feedback: Instant green success highlight when a coupon is applied correctly.
UX Weaknesses
  • Cart Persistence: Users have to navigate back to "Shop" to continue; needs a Side-Drawer cart.
  • Address Friction: Manual typing of state/city is slow; needs Autocomplete.
UX FIX: Side-Cart Drawer

Implement a slide-out cart drawer instead of a full-page cart to keep the user in the "shopping context."

FLOW: SECURITY

Authentication Flow Audit

Mapping the move to Passwordless (OTP) Login.

Pros (Passwordless)
  • Zero "Forgot Password" tickets (massive UX win).
  • Secure verification via valid email/phone.
  • Frictionless mobile experience (no typing complex passwords).
Cons
  • Requires "Email Send" overhead; if email lags, flow fails.
  • New users might be confused why there is no "Password" field.
UX FIX: Educational Tooltips

Add a small info label: "Login securely with a 6-digit code sent to your email. No password needed."
